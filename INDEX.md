## 📱 Quick Reference

### **Platforms**
- Frontend: http://localhost:3000
- Backend: http://localhost:8000
- Admin: http://localhost:8000/admin
- API Schema: http://localhost:8000/api/docs

### **Key Files**
- Backend: `backend/config/settings.py`, `apps/`
- Frontend: `frontend/src/App.js`, `Dashboard.jsx`
- Database: `backend/db.sqlite3`
- Tests: `test_*.csv` files

### **Main Classes**
- `Company`, `CompanyUser` (Multi-tenancy)
- `DataSource`, `IngestionJob`, `RawRecord` (Ingestion)
- `ActivityRecord`, `AuditLog` (Normalization)
- `ReviewSession`, `ReviewQueue` (Review Workflow)

### **API Endpoints**
- `POST /api/ingestion/jobs/upload/` - Upload CSV
- `GET /api/review/records/` - List records
- `POST /api/review/records/{id}/approve/` - Approve
- `GET /api/review/records/statistics/` - Get stats
