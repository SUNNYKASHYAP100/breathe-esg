## 🚨 If You See Errors

### "Failed to load data: Request failed with status code 500"
**Fix:**
1. Check backend is running: Visit https://breathe-esg-uczd.onrender.com/api
2. Check CORS: Verify frontend URL in CORS_ALLOWED_ORIGINS
3. Check database: PostgreSQL connected & migrations applied
4. Check logs: Render dashboard → Logs tab

### "Cannot connect to API"
**Fix:**
1. Verify REACT_APP_API_URL environment variable is set
2. Check frontend API URL in .env file
3. Rebuild & redeploy frontend
4. Check network tab in browser DevTools

### "Database connection error"
**Fix:**
1. Verify DATABASE_URL environment variable
2. Check PostgreSQL is running on Render
3. Verify username & password
4. Run migrations: `python manage.py migrate`
